စာရိုက်အမြန်နှုန်းစမ်းသပ်မှု - WPM၊ Net WPM နှင့် တိကျမှု

သင်ရိုက်မည်မျှမြန်ပြီး မည်မျှတိကျသည်ကို တိုင်းတာပါ။ အင်္ဂလိပ် သို့မဟုတ် တရုတ်နမူနာကို ရွေးပါ၊ စာရိုက်စတင်ပါ၊ ကိရိယာသည် သင်၏အချိန်၊ စုစုပေါင်း WPM၊ အသားတင် WPM နှင့် တိကျမှုကို တိုက်ရိုက်ခြေရာခံပါ။ အရာအားလုံးသည် သင့်ဘရောက်ဆာတွင် စက်တွင်းလည်ပတ်သည်။ သင်ရိုက်သောစာသားကို ဘယ်တော့မှ အပ်လုဒ်မလုပ်ပါ။

စာမေးပွဲ ဘယ်လိုအလုပ်လုပ်လဲ။

နမူနာတစ်ခုကို ရွေးပါ၊ ထို့နောက် ၎င်းကို မြန်နိုင်သမျှ တိကျစွာ ရိုက်ထည့်ပါ။ အချိန်တိုင်းကိရိယာသည် သင်၏ပထမဆုံးသော့ချက်စကုတ်ဖြင့် စတင်သည်။ သင်ရိုက်သည့်အတိုင်း၊ စာလုံးတစ်လုံးစီကို နမူနာနှင့် နှိုင်းယှဉ်သည်- ကိုက်ညီမှုများသည် သင့်အမြန်နှုန်းအတွက် ရေတွက်ပြီး မကိုက်ညီမှုများကို အမှားများအဖြစ် တွက်ချက်ပါသည်။ နမူနာ၏အဆုံးကို သင်ရောက်ရှိသောအခါ၊ သင်၏နောက်ဆုံး WPM နှင့် တိကျမှုသည် အေးခဲသွားပါသည်။

WPM နှင့် တိကျမှုကို မည်သို့တွက်ချက်သနည်း။

စံကွန်ဗင်းရှင်းသည် စကားလုံးတစ်လုံးကို နေရာလွတ်များအပါအဝင် သော့စထရိုက်ငါးခုအဖြစ် ရေတွက်သည်။ စုစုပေါင်း WPM = (စာလုံးရိုက်ပြီး / 5) / မိနစ်အကြာ။ Net WPM သည် ပြုပြင်မထားသော အမှားများကို နုတ်သည်- Net WPM = စုစုပေါင်း WPM - (အမှားများ / မိနစ်)။ တိကျမှု = မှန်ကန်သော စာလုံးများ / စုစုပေါင်းရိုက်ထည့်ထားသော x 100%။ တရုတ်စာအတွက်၊ တရုတ်စာလုံးတစ်လုံးသည် လက်တင်ကီးစထရိတစ်ခုထက် သတင်းအချက်အလက်ပိုပေးသောကြောင့် တစ်မိနစ်လျှင် အက္ခရာများအဖြစ် အမြန်နှုန်းကို မကြာခဏ အစီရင်ခံပါသည်။

စာရိုက်မြန်နှုန်း စံနှုန်းများ (2026 ခန့်မှန်းချက်) 2026

သင့်ရမှတ်ကိုအဓိပ္ပာယ်ပြန်ဆိုရန် ဤအကိုးအကားအပိုင်းအခြားများကိုသုံးပါ။ ၎င်းတို့သည် ပထမပါတီတိုင်းတာခြင်းမဟုတ်ဘဲ အများသူငှာ စာရိုက်မြန်နှုန်းဒေတာမှ ပေါင်းစပ်ထားသော ခန့်မှန်းချက်ဖြစ်သည်။

အဆင့်ပုံမှန် WPMကမ္ဘာ့ကျန်းမာရေးအဖွဲ့
Beginner/bill-and-peck~27 WPMကီးဘုတ်ကိုကြည့်နေသော လက်နှစ်ချောင်းစာရိုက်သမား
ပျမ်းမျှ ကွန်ပျူတာ အသုံးပြုသူ~40 WPMနေ့စဉ်အသုံးပြုသူအများစု
ပရော်ဖက်ရှင်နယ် စာစီစာရိုက်သမား65-75 WPMဒေတာထည့်သွင်းခြင်းနှင့် စာသားမှတ်တမ်းလုပ်ခြင်း
အဆင့်မြင့်/အပြိုင်အဆိုင်100-120+ WPMစာစီစာရိုက်သူ၊ ဂိမ်းကစားသူ၊ ပရိုဂရမ်မာများကို ထိပါ။
မှတ်တမ်းကိုင်ဆောင်သူများ150-212 WPMစဉ်ဆက်မပြတ် ကမ္ဘာ့စံချိန်များ (ဥပမာ Barbara Blackburn)

ခန့်မှန်းချက်၊ ပထမပါတီတိုင်းတာခြင်းမဟုတ်ပါ။ အများသူငှာ ကိုးကားချက်များမှ စုစည်းထားခြင်း၊ တစ်ဦးချင်းစီ ရလဒ်များသည် ကီးဘုတ်၊ ဘာသာစကားနှင့် အပြင်အဆင်တို့နှင့် ကွဲပြားသည်။

အရင်းအမြစ်များ: Wikipedia - Words per minute · Wikipedia - Typing

မေးလေ့ရှိသောမေးခွန်းများ

ကောင်းမွန်တဲ့ စာရိုက်နှုန်းက ဘာလဲ။

40 WPM သည် နေ့စဉ် ကွန်ပျူတာ သုံးစွဲသူများအတွက် ပျမ်းမျှဖြစ်သည်။ 65-75 WPM သည် သာမန် ပရော်ဖက်ရှင်နယ် စံနှုန်းတစ်ခုဖြစ်ပြီး လက်နှိပ်စက် ကိုင်သူများသည် 100 WPM သို့မဟုတ် ထို့ထက်ပို၍ ရောက်တတ်သည်။ ခန့်မှန်းချက်နှင့်အတူ တရုတ်ပင်ယင်ထည့်သွင်းမှုအတွက်၊ လူများသည် WPM ထက် တစ်မိနစ်လျှင် စာလုံးများကို နှိုင်းယှဉ်လေ့ရှိသည်။

WPM ဆိုတာ ဘာကို ဆိုလိုတာလဲ။

တစ်မိနစ်လျှင် စကားလုံးများ။ ကွန်ဗင်းရှင်းအရ စကားလုံးတစ်လုံးသည် နေရာလွတ်များအပါအဝင် သော့ချက်စထရိငါးခုနှင့် ညီမျှသည်၊ ထို့ကြောင့် WPM = (ဇာတ်ကောင်များ / 5) / မိနစ်။

net WPM ဆိုတာဘာလဲ။

Net WPM သည် တစ်မိနစ်လျှင် သင်၏ စုစုပေါင်း WPM အနုတ်လက္ခဏာဆောင်သော အမှားများဖြစ်သည်။ ၎င်းသည် ကုန်ကြမ်းအမြန်နှုန်းတင်မဟုတ်ဘဲ တိကျမှုကို ပေးသည်။

ကျွန်ုပ်စာရိုက်ခြင်းကို မည်သည့်နေရာတွင်မဆို အပ်လုဒ်လုပ်ထားပါသလား။

နံပါတ်၊ စစ်ဆေးမှုသည် သင့်ဘရောက်ဆာတွင် လုံး၀ လုပ်ဆောင်သည်။ နမူနာစာသားနှင့် သင်ရိုက်ထည့်သောအရာသည် သင့်စက်ကို ဘယ်သောအခါမှ မထားခဲ့ပါ။

တရုတ်လို ဘယ်လိုမြန်အောင်ရိုက်ရမလဲ။

ပင်ယင်သော့များအတွက် ထိထိရိုက်နှိပ်ခြင်းကိုသုံးပါ၊ ဘုံစကားစုဖြတ်လမ်းများကိုလေ့လာပါ၊ နှင့် cloud ခန့်မှန်းချက်သည် ရှည်လျားသောစကားလုံးများကို ပြီးမြောက်စေပါ။ Sogou Input Method desktop client သည် cloud input နှင့် စကားစု ခန့်မှန်းခြင်းကို ပေးဆောင်သည်။

စမ်းသပ်မှုသည် တရုတ်နမူနာစာသားကို ပံ့ပိုးပေးပါသလား။

ဟုတ်ကဲ့။ တရုတ်စာရိုက်ခြင်းကို လေ့ကျင့်ရန် တရုတ်နမူနာသို့ ပြောင်းပါ၊ သို့မဟုတ် လက်တင်-စာရိုက်နှုန်းအတွက် အင်္ဂလိပ်နမူနာသို့ ပြောင်းပါ။
