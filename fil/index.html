Sukatin kung gaano kabilis at kung gaano ka tumpak ang pagta-type mo. Paano gumagana ang pagsubok

Pumili ng sample, pagkatapos ay i-type ito nang mabilis at tumpak hangga't maaari. Magsisimula ang timer sa iyong unang keystroke. Habang nagta-type ka, inihahambing ang bawat character sa sample: ang mga tugma ay binibilang sa iyong bilis at ang mga hindi pagkakatugma ay binibilang bilang mga error. Kapag naabot mo ang dulo ng sample, ang iyong panghuling WPM at katumpakan ay na-freeze.

Paano kinakalkula ang WPM at katumpakan

Ang karaniwang convention ay binibilang ang isang salita bilang limang keystroke, kabilang ang mga puwang. Gross WPM = (mga character na na-type / 5) / lumipas ang mga minuto. Binabawas ng Net WPM ang mga hindi naitama na error: Net WPM = Gross WPM - (mga error / minuto). Katumpakan = tamang mga character / kabuuang na-type x 100%. Para sa Chinese, ang bilis ay madalas na iniuulat bilang mga character kada minuto, dahil ang isang Chinese na character ay nagdadala ng higit pang impormasyon kaysa sa isang Latin na keystroke.

Mga madalas itanong

Ano ang magandang bilis ng pag-type?

Nasa 40 WPM ang karaniwan para sa pang-araw-araw na mga gumagamit ng computer. Ang 65-75 WPM ay isang karaniwang propesyonal na benchmark, at ang mga touch typist ay kadalasang umaabot sa 100 WPM o higit pa. Para sa Chinese pinyin input na may hula, ang mga tao ay karaniwang naghahambing ng mga character bawat minuto sa halip na WPM.

Ano ang ibig sabihin ng WPM?

Mga salita kada minuto. Sa pamamagitan ng convention ang isang salita ay katumbas ng limang keystroke kabilang ang mga puwang, kaya WPM = (character / 5) / minuto.

Ano ang net WPM?

Ang Net WPM ay ang iyong kabuuang WPM na binawasan ng mga hindi naitama na error bawat minuto. Ginagantimpalaan nito ang katumpakan, hindi lamang ang hilaw na bilis.

Na-upload ba ang aking pagta-type kahit saan?

Hindi. Ang pagsubok ay ganap na tumatakbo sa iyong browser; ang sample na text at kung ano ang tina-type mo ay hindi kailanman umaalis sa iyong device.

Paano ako makakapag-type ng mas mabilis sa Chinese?

Gumamit ng touch typing para sa mga pinyin key, alamin ang mga karaniwang shortcut ng parirala, at hayaang kumpletuhin ng hula sa ulap ang mahahabang salita. Nag-aalok ang Sogou Input Method desktop client ng cloud input at hula ng parirala.

Sinusuportahan ba ng pagsubok ang sample na teksto ng Chinese?

Oo. Lumipat sa Chinese sample para magsanay ng Chinese type, o lumipat sa English na sample para sa Latin-script na bilis.
